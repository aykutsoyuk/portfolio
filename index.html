Rainbow's BFF
Github Repo Visit The Website

Rainbow’s B.F.F. is a full-stack web application project with API support for authentication, and storage of desired color palettes generated (many thanks to M. Ege Ercan for the back-end side). The user can chose the generate method. Each color shows their hex codes and names. When signed in, generated colors can be saved in user’s palette library. The user can delete or update their saved palettes. And previously created palettes can be reached.

If you visit the repo, you will be informed of some topics about that process.
